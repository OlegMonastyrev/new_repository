## Списки

Чтобы добавить нумерованные списки, требуется просто пронумеровать пункты, например:

1. элемент номер 1
2. элемент номер 2 
3. элемент номер 3

Чтобы добавить ненумерованные списки, требуется выделить пункты символом (*), например:

* элемент номер 1
* элемент номер 2 
* элемент номер 3
